## 项目数据集
项目主要采用COCO2017与AI Challenge两个数据集进行训练与评估


([AI Challenger数据集](https://ai.chuangxin.com/ai_challenger))是一个大规模中文视觉描述数据集，主要特点包括：
- 每张图片配有5句精准的中文描述
- 总规模：30万张高质量图片，150万条标注文本
- 数据划分：
  - 训练集：210,000张(70%)
  - 验证集：30,000张(10%) 
  - 测试集A/B：各30,000张(各10%)
